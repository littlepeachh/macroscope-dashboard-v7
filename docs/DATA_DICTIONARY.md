# 指标与单位字典

| 指标 | 公式/口径 | 单位 |
|---|---|---|
| M1、M2余额 | 公开月度货币供应量 | 万亿元 |
| M1、M2同比 | 当月同比增速 | % |
| M1-M2剪刀差 | M1同比 - M2同比 | 百分点 |
| 社融增量 | 当月社会融资规模增量 | 万亿元 |
| 社融存量同比 | 社融存量同比增速 | % |
| DR001、DR007 | 存款类机构质押式回购加权利率 | % |
| 隔夜Shibor | 隔夜上海银行间同业拆放利率 | % |
| 美国国债收益率 | DGS2、DGS10 | % |
| 交易拥挤度 | 成交额前5%股票成交额 / 全部沪深A股成交额 | % |
| 上涨/下跌家数 | 当日涨跌幅大于/小于0的A股数量 | 只 |
| 两市成交额 | 沪深A股成交额合计 | 万亿元 |
| A股广义换手率 | 两市成交额 / A股总市值 | % |
| 两融余额 | 沪深融资融券余额合计 | 万亿元 |
| 两融余额/总市值 | 两融余额 / A股总市值 | % |
| 10周偏离度 | 指数收盘 / 10周均线 - 1 | % |
| 20周偏离度 | 指数收盘 / 20周均线 - 1 | % |
| 单只基金募集份额 | 新成立基金公开募集份额 | 亿份 |
| 估算募集规模 | 募集份额×1元/份近似 | 亿元 |

## 颜色

所有市场统一：

- 红色：上涨、正收益、正偏离。
- 绿色：下跌、负收益、负偏离。
